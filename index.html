¡Modo Plantilla Activado!

  1. **Sube tu Diseño:** Este código espera que subas tu diseño de Graphionica como **plantilla_bingo.png** a la raíz de tu proyecto.
  2. **Diseño 2x2:** Crea tu imagen PNG con el diseño de las 4 cartas, incluyendo el fondo, logo, y las letras BINGO. La proporción ideal para 700px de ancho es de **700px x 990px** para una relación A4.
  3. **Alineación:** Si los números generados no caen exactamente donde quieres, ajusta el padding del elemento #generated-content-wrapper y .high-res-capture #generated-content-wrapper en la sección style.
  4. **Edición de Color:** Si el color de los números (#000000) o la ID no contrasta bien con tu fondo, ajusta las variables --card-number-color y --id-text-color al inicio del CSS.
